## PROJECT DESCRIPTION

This project is a responsive personal portfolio website built using
HTML5 and CSS3. It showcases personal information, technical skills,
featured projects, and contact information in a clean, modern layout.

The website is intended to serve as an online portfolio for internship,
freelance, and job opportunities.
